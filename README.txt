#Decision Trees
An implementation of decision trees for regression and classification in Java

Assignment written in Eclipse for Java, Version: 2019-12 (4.14.0)

The program for Intro to Machine Learning Programming Assignment 3 can be run as follows:

>java ML3 abalone.data breast-cancer-wisconsin.data car.data forestfires.csv machine.data segmentation.data

Output files for each input dataset will be written as '<input dataset name>.out.txt'. Output contains summary statistics, details of runs are printed to console.
